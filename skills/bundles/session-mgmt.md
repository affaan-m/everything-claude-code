# Session & Learning Bundle

Tools for managing session state, extracting patterns, and optimizing context usage.

## Skills
1. **save-session** — Save current session state for later resumption
2. **resume-session** — Load most recent session and resume with full context
3. **sessions** — Manage session history, aliases, and metadata
4. **checkpoint** — Create session checkpoint at logical intervals
5. **continuous-learning-v2** — Instinct-based learning: observe sessions, create atomic instincts, evolve into skills
6. **instinct-status** — Show learned instincts (project + global) with confidence scores
7. **instinct-import** / **instinct-export** — Move instincts between scopes
8. **learn** — Extract reusable patterns from current session
9. **learn-eval** — Extract patterns + self-evaluate quality before saving
10. **knowledge-consolidation** — Build frameworks from scattered insights across braindumps
11. **context-budget** — Analyze context window usage, find optimization opportunities
12. **strategic-compact** — Manual context compaction at logical intervals
13. **kb-workflow** — Search knowledge base before tasks, capture findings after
14. **ck** — Persistent per-project memory, auto-loads context on session start
