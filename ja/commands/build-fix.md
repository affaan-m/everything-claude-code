# ビルドエラー修正

TypeScript およびビルドエラーを段階的に修正：

1. ビルドを実行：npm run build または pnpm build

2. エラー出力を解析：
   - ファイルごとにグループ化
   - 重要度順にソート

3. 各エラーについて：
   - エラーのコンテキストを表示（前後5行）
   - 問題を説明
   - 修正を提案
   - 修正を適用
   - ビルドを再実行
   - エラーが解決したことを確認

4. 以下の場合は停止：
   - 修正が新たなエラーを引き起こした場合
   - 同じエラーが3回の試行後も解決しない場合
   - ユーザーが一時停止を要求した場合

5. サマリーを表示：
   - 修正されたエラー
   - 残存するエラー
   - 新たに発生したエラー

安全のためエラーは1つずつ修正すること！
