# Orchestration Bundle

Tools for coordinating multiple agents, running autonomous loops, and team-based execution.

## Skills
1. **dispatch** — Decompose tasks and route subtasks to optimal agent pipelines
2. **orchestrate** — Sequential and tmux/worktree guidance for multi-agent workflows
3. **team-builder** — Interactive agent picker for composing parallel teams
4. **devfleet** / **claude-devfleet** — Plan projects, dispatch agents in isolated worktrees, monitor progress
5. **autonomous-loops** — Patterns from simple pipelines to RFC-driven multi-agent DAGs
6. **continuous-agent-loop** — Autonomous loops with quality gates, evals, recovery controls
7. **loop** — Run a prompt or command on a recurring interval
8. **dmux-workflows** — Multi-agent orchestration using tmux pane manager
9. **ralphinho-rfc-pipeline** — RFC-driven DAG execution with quality gates and merge queues
10. **enterprise-agent-ops** — Long-lived agent workloads with observability and security boundaries
11. **omc-reference** — OMC agent catalog, tools, team pipeline routing, commit protocol
12. **blueprint** — One-line objective → multi-session construction plan with adversarial review
13. **multi-plan** / **multi-execute** / **multi-workflow** — Multi-model collaborative modes
14. **autonomous-agent-harness** — Persistent memory, scheduled ops, task queue for autonomous agents
15. **gan-style-harness** — GAN-inspired Generator-Evaluator agent harness for quality output
