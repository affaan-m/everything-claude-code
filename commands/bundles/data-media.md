# Data & Media Bundle

Tools for working with media, documents, data scraping, and NotebookLM integration.

## Skills
1. **videodb** — Ingest, index, search video/audio from files, URLs, RTSP, or live feeds
2. **video-editing** — AI-assisted video editing: FFmpeg, Remotion, ElevenLabs, fal.ai, Descript
3. **fal-ai-media** — Image, video, audio generation via fal.ai (Seedance, Kling, Veo 3, CSM-1B)
4. **data-scraper-agent** — Automated data collection from any public source on schedule
5. **nutrient-document-processing** — PDF/DOCX/XLSX processing, OCR, redaction, signing
6. **notebooklm-skill** — Query NotebookLM for source-grounded, citation-backed answers
7. **notebooklm-py** — Full programmatic NotebookLM API access (create notebooks, generate artifacts)
8. **anything-to-notebooklm** — Multi-source content processor: web, YouTube, PDF → NotebookLM
9. **remotion-video-creation** — Remotion best practices for video creation in React (3D, animations, audio, captions)
10. **ui-demo** — Record polished UI demo videos using Playwright

## MCP servers
- **notebooklm** — NotebookLM integration
- **firecrawl** — Web scraping and crawling
