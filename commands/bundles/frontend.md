# Frontend & Design Bundle

Activate ALL of these skills for this task. Each skill adds a layer of design intelligence:

## Skills to activate (in order)

1. **ui-ux-pro-max** — Style direction, palette (161 options), font pairing (57 options), UX guidelines (99 rules)
2. **frontend-design** — Production-grade UI implementation, anti-generic patterns
3. **web-designer** — Scrollytelling, GSAP, Three.js, immersive experiences
4. **web-design-guidelines** — Web Interface Guidelines compliance check
5. **ui-styling** — shadcn/ui components + Tailwind patterns
6. **design-system** — Token architecture, component specs, spacing scales
7. **design** — Brand identity, logos, banners, social assets
8. **frontend-patterns** — React/Next.js/Vue state management, performance
9. **coding-standards** — TypeScript/JavaScript quality standards
10. **e2e-testing** — Playwright patterns for visual regression
11. **nextjs-turbopack** — Next.js 16+ with Turbopack, incremental bundling, dev speed
12. **nuxt4-patterns** — Nuxt 4 hydration safety, performance, route rules, SSR-safe data fetching
13. **bun-runtime** — Bun as runtime/bundler/test runner, migration notes, Vercel support
14. **browser-qa** — Visual testing and UI interaction verification via browser automation
15. **ui-demo** — Record polished UI demo videos with Playwright

## Also use these MCP servers
- **@21st-dev/magic** — Search real component examples FIRST
- **figma** — Pull designs from Figma if available
- **playwright** — Screenshot for visual verification

## Anti-vibecode checklist (from rules/web/design-quality.md)
Before writing ANY component:
- [ ] Picked a style direction (not "clean minimal")
- [ ] Selected palette from ui-ux-pro-max
- [ ] Chose font pairing with character
- [ ] Searched @21st-dev/magic for reference components
- [ ] Design has 4+ of: hierarchy, depth, typography, color, micro-interactions, layout, texture, motion
