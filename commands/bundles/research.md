# Research Bundle

Activate ALL research skills and MCPs for comprehensive investigation:

## Skills
1. **deep-research** — Multi-source research with firecrawl + exa
2. **exa-search** — Neural search for web, code, companies
3. **auto-research** — Parallel research threads with synthesis
4. **market-research** — Competitive analysis, industry intelligence
5. **notebooklm-skill** — Query NotebookLM for grounded answers from your docs
6. **notebooklm-py** — Programmatic NotebookLM access
7. **search-first** — Always search before coding
8. **kb-workflow** — Check knowledge base before and after work
9. **documentation-lookup** — Up-to-date library/framework docs via Context7 MCP

## MCP servers to use
- **exa** — Neural web search
- **firecrawl** — Deep web scraping
- **notebooklm** — Document-grounded research
- **context7** — Library/API documentation
- **github** — Code search across GitHub

## Research protocol
1. Define research questions clearly
2. Search knowledge base first (kb-workflow)
3. Search NotebookLM for existing docs
4. Fan out to web (exa + firecrawl) in parallel
5. Cross-reference findings
6. Synthesize with citations
7. Save findings to knowledge base
