# ビルドと修正

TypeScriptとビルドエラーを段階的に修正:

1. ビルドを実行: `npm run build` または `pnpm build`

2. エラー出力を解析:
   - ファイル別にグループ化
   - 重大度でソート

3. 各エラーについて:
   - エラーのコンテキストを表示（前後5行）
   - 問題を説明
   - 修正を提案
   - 修正を適用
   - ビルドを再実行
   - エラーが解決されたことを確認

4. 以下の場合は停止:
   - 修正が新しいエラーを引き起こした
   - 同じエラーが3回の試行後も続く
   - ユーザーが一時停止を要求

5. サマリーを表示:
   - 修正したエラー
   - 残っているエラー
   - 新たに発生したエラー

安全のために一度に1つのエラーを修正！
