### `hooks/` — 이벤트 기반 자동화

도구 실행 전후, 세션 시작/종료 시 자동으로 실행되는 스크립트입니다.

```
hooks/
├── hooks.json          # 훅 전체 설정 (매처 조건 + 명령어 정의)
└── README.md           # 훅 시스템 설명
```

**훅 스크립트는 `scripts/hooks/`에 위치합니다:**

```
scripts/hooks/
├── session-start.js         # 세션 시작 시 컨텍스트 초기화
├── session-end.js           # 세션 종료 시 요약 저장
├── post-edit-format.js      # 파일 편집 후 자동 포맷팅
├── post-edit-typecheck.js   # 파일 편집 후 타입 검사
├── pre-bash-commit-quality.js  # 커밋 전 품질 검사
├── block-no-verify.js       # --no-verify 플래그 차단
├── cost-tracker.js          # API 비용 추적
├── quality-gate.js          # 품질 게이트 검사
├── run-with-flags.js        # 훅 프로파일/비활성화 래퍼
└── ...                      # 기타 pre/post 훅들
```

> **훅 실행 환경 조절:**
> ```bash
> export ECC_HOOK_PROFILE=minimal   # minimal | standard | strict
> export ECC_DISABLED_HOOKS="pre:bash:tmux-reminder,post:edit:typecheck"
> ```

| MCP 설정 | JSON | — |

## 설치 프로파일

```bash
# 전체 설치 (권장)
./install.sh --profile full

# 특정 언어만 설치
./install.sh typescript python golang

# 특정 하네스 대상으로 설치
./install.sh --target cursor typescript
./install.sh --target gemini --profile full
```
